# 📽️ 추천 알고리즘 콘텐츠 PT 프로젝트

## 📌 프로젝트 정보
- 기간 : 2023.05 ~ 2023.06  
- 기관 : 동양미래대학교  
- 기여도 : 100%

## 📖 프로젝트 소개
추천 시스템의 핵심 알고리즘 구조와 동작 원리를 학습자 관점에서 쉽게 이해할 수 있도록  
콘텐츠 형태로 시각화하여 발표한 프로젝트입니다.

사용자 맞춤 추천 알고리즘의 정의, 활용 사례, 동작 방식, 기술 구성 요소 등을  
직관적인 그래픽과 예시를 통해 설명하는 PT 자료를 제작하였습니다.

## 🧑‍💻 담당 역할
- 추천 알고리즘 원리 조사 및 구조 정리  
- 사례 기반 설명 콘텐츠 기획  
- 시각 자료(PPT) 제작 및 발표 수행  

## 📚 Lesson Learned
- 추천 시스템 구조 전반에 대한 이해 향상  
- 복잡한 기술 개념을 쉽게 전달하는 시각화 역량 강화  
- 발표를 통한 커뮤니케이션 능력 향상
